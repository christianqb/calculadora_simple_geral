# 🧮 Calculadora Simples em JavaScript

Este projeto foi desenvolvido com o objetivo de **revisar e praticar conceitos básicos de JavaScript**, após um período sem programar.

A ideia foi criar uma calculadora funcional e simples, reforçando fundamentos importantes da linguagem e da lógica de programação.

---

## 🚀 Funcionalidades

* Operações básicas:

  * ➕ Soma
  * ➖ Subtração
  * ✖️ Multiplicação
  * ➗ Divisão
* Interface simples e intuitiva
* Atualização dinâmica do resultado

---

## 🧠 Conceitos praticados

Durante o desenvolvimento, foram revisados:

* Variáveis 
* Operadores matemáticos
* Manipulação do DOM
* Eventos (`click`)
* Funções
* Organização de código

---
